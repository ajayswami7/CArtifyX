# About

**CArtifyX** is a modern luxury fashion shopping website inspired by premium fashion brands and leading e-commerce platforms. The application provides customers with an elegant shopping experience while offering administrators powerful tools to manage products, categories, and customer orders.

The project demonstrates complete **full-stack web development** using **PHP**, **MySQL**, **HTML**, **CSS**, **JavaScript**, and **Bootstrap 5**.

*The complete source code is private. It can be shared upon request for academic or recruitment purposes*

# Installation

## 1. Clone Repository

```bash
git clone https://github.com/ajayswami7/CArtifyX.git
```

---

## 2️. Move Project

Copy the project folder into

```
xampp/htdocs/
```

---

## 3️. Start XAMPP

Start

- Apache
- MySQL

---

## 4️. Import Database

Open

```
http://localhost/phpmyadmin
```

Create database

```
cartifyx
```

Import

```
database/cartifyx.sql
```

---

## 5️. Configure Database

Open

```
includes/config.php
```

Update

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "cartifyx";
```

---

## 6️. Run Project

Open

```
http://localhost/CArtifyX/
```
