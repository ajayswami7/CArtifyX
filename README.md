# CArtifyX

Luxury Fashion Shopping Website

## Features

- User Login
- Admin Dashboard
- Product Search
- Wishlist
- Shopping Cart
- Checkout
- Responsive Design

## Technologies

- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- MySQL

## Installation

1. Clone repository

```bash
git clone https://github.com/ajayswami7/CArtifyX.git
```

2. Import database

```
database/cartifyx.sql
```

3. Start XAMPP

4. Visit

```
http://localhost/CArtifyX
```
